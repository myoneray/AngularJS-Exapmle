ng-src

将 ng-src 指令换成普通的 src 属性。用像Firebug，Chrome Web Inspector这样的工具，
或者直接去看服务器的访问日志， 你会发现你的应用向 /app/%7B%7Bphone.imageUrl%7D%7D （或者/app/{{phone.imageUrl}}）发送了一个非法请求
这个问题是由于浏览器会在遇到 img 标签的时候立刻向还未得到编译的URL地址发送一个请求，
AngularJS只有在页面载入完毕后才开始编译表达式从而得到正确的图片URL地址。
